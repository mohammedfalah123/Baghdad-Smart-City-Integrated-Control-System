# Security Policy

## Supported Versions
Currently, the following versions of the Baghdad Smart City Control System receive security updates:
| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅ Yes     |

## Reporting a Vulnerability
We are committed to the security of smart city energy management. If you discover a security vulnerability (especially in the 70 algorithms or the control logic), please do not disclose it publicly.

Instead, please report it through one of the following methods:
1. **GitHub Issues:** Open a private issue if possible, or a public one for non-sensitive bugs.
2. **Email:** [ضع إيميلك هنا]

We will acknowledge your report within 48 hours and provide a timeline for fixing the issue.

## Security Audits
This project follows the **OpenSSF Best Practices** and undergoes automated code scanning via **GitHub CodeQL** to ensure the integrity of its 70 algorithms.
